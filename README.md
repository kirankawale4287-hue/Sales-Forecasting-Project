# Sales Forecasting Using Time Series Analysis

## Project Objective
To predict future sales using historical time-series sales data.

## Dataset
Sample retail sales dataset with Date and Sales values.

## Tools Used
- Python
- Pandas
- Matplotlib
- Statsmodels
- Kaggle

## Methodology
1. Data loading and preprocessing
2. Visualization of historical sales
3. ARIMA model training
4. Future sales prediction
5. Actual vs Predicted visualization

## Result
The ARIMA model successfully predicted future sales trends based on historical data.

## Future Scope
- Use LSTM deep learning model
- Use real-world retail dataset
- Build web dashboard using Streamlit
